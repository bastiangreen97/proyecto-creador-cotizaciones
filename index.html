Proyecto creador de cotizaciones

Este proyecto tiene como objetivo brindar una pequeña ayuda a pymes y emprendedores que no cuenten con un sistema que les permita realizar cotizaciones, facilitando la creación de estas, cabe destacar que este sistema no reemplaza a uno de ventas, ni la cotización a una boleta.

Comienza registrando tus productos y/o servicios para poder utilizarlos en tus futuras cotizaciones

Crea cotizaciones según lo solicitado por tus clientes, puedes combinar servicios con productos, o solo hacer de un tipo

Una vez que termines de crear una cotización puedes descargarla en formato PDF para ser impresa o enviada por distintos medios como whatsapp, correo, messenger, etc. a tu cliente
