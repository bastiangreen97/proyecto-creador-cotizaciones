.- Inicio -.

Este proyecto tiene como objetivo brindar una pequeña ayuda a pymes y emprendedores, a través de la creación de cotizaciones de forma digital. El sitio consta de 2 secciones:

Gestión de servicios y productos: Aquí puedes registrar tus productos y/o servicios con sus respectivos datos, también puedes eliminarlos o editarlos. Es necesario tener al menos un registro para realizar una cotización

Creador de cotizaciones: Aquí puedes crear tus cotizaciones, para realizar una cotización consta de 3 sencillos pasos:

  • 1. Ingresar tus datos y el nombre del cliente.
  • 2. Agregar productos/servicios a la cotización para generar el calculo Total.
  • 3. Descargar PDF de la cotización con todos los datos ingresados anteriormente.

Los datos registrados son almacenados de forma local y en un tiempo indefinido en tu navegador web (Ejemplo: Google Chrome, Opera, etc) hasta que se limpien/borren los datos del sitio en tu navegador web.

Comienza registrando tus productos y/o servicios para utilizarlos en tus futuras cotizaciones

Crea cotizaciones según las solicitudes de tus clientes, dentro de una cotización puedes agregar los productos y servicios que has registrado

Cuando termines de crear una cotización no olvides descargarla en formato PDF, para que puedas enviarla a tu cliente vía Whatsapp, email, messenger, etc
